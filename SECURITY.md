# Security Policy - MalinWallet

MalinWallet is committed to the security of our users' assets. We take all security reports seriously.

## Supported Versions

Only the latest version of MalinWallet is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 7.2.x   | :white_check_mark: |
| < 7.1   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in MalinWallet, please report it immediately via email:

**Email:** contact@w-malin.tech

Please **do not** create a public GitHub issue for security vulnerabilities. 

### What we expect from your report:
- A detailed description of the vulnerability.
- Steps to reproduce the issue (PoC).
- Any potential impact on user funds or privacy.

## Our Commitment
We will acknowledge receipt of your report within 24 hours and provide a timeline for a fix if the vulnerability is confirmed.

*Note: This is a Non-Custodial wallet. Users are responsible for their own seed phrases.*
